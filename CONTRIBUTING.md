# Contributing

Thanks for contributing to AirPods Battery for Windows.

## Development setup

1. Install Python 3.10+.
2. Install dependencies:
   - `pip install -r requirements.txt`
3. Run locally:
   - `python -m src.main`

## Build the EXE

- Run:
  - `.\build.bat`
- Output is generated at:
  - `dist/AirPodsBattery.exe`

## Pull requests

- Keep PRs focused and small.
- Add a short description of what changed and why.
- Include manual test notes (what you ran and observed).

## Code style

- Follow existing structure and naming patterns.
- Keep comments concise and only where needed.
- Prefer clear, readable code over clever code.
